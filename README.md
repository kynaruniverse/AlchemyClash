# ⚡ FUSIONGOD ⚡
**A Modular AAA Dark-Fantasy Card Crawler**  
Built with Phaser v3.90.0 "Tsugumi" — fully refactored for scalability.

## ✅ Current Status (Phase 6 Complete)
- Fully modular architecture (EventBus + Managers + Entities)
- Clean data-driven design
- AAA juice (particles, shake, pop animations, haptics)
- Robust save system + error handling
- Mobile-optimized (Spck Editor + Vercel)

**Live Demo:** [https://fusion-god.vercel.app/](https://fusion-god.vercel.app/)

## 📂 Project Structure
core/          → EventBus + Constants
data/          → All game data (elements, dungeon, fusions)
entities/      → Card class
managers/      → Hand, UI, Fusion, Dungeon
systems/       → ParticleSystem + SaveSystem

## 🛠️ Tech Stack
- Phaser 3.90.0
- ES6 Modules + EventBus pattern
- Vercel deployment

## 🚀 How to Play
1. Drag cards onto each other → **Fuse**
2. Drag cards to dungeon row → **Fight / Loot / Trap**
3. Survive and discover new hybrids!
