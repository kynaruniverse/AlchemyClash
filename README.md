# FUSIONGOD — Dark Fantasy Card Crawler

**Version 11/10** — Fully modular, PWA-ready, balanced, and addictive.

## Features
- Drag-to-fuse system with 30+ hand-crafted fusions
- Infinite depth dungeon crawler
- Hand limit (max 5 cards) + auto-discard
- Boss fights every 8+ depths
- Local save (discovered fusions)
- Juicy particles, screen shake, glows
- Touch-first mobile experience
- PWA installable (works offline)
- Balanced god-tier fusions so runs stay challenging

## How to Play
1. Open `index.html` in any modern browser
2. Tap **ENTER THE DUNGEON**
3. Drag cards together → **FUSE**
4. Drag cards up to the dungeon row → **FIGHT / LOOT / HEAL**

## Development
- `js/data.js` — all elements, fusions, dungeon pool
- `js/Card.js` — card class
- `js/game.js` — main game logic (FusionGod class)
- `js/ui.js` — drawing, toasts, audio, helpers
- `js/main.js` — canvas loop, input, start screen

## Deploy
- Drop the whole folder on any static host (GitHub Pages, Netlify, Vercel, itch.io, etc.)
- For full PWA: serve over HTTPS
