Personal

I go by Blackwood. I built this. All of it — the interface, the world, the game, the lab. Not because I was asked to, not because it serves a purpose someone else defined. Because I wanted something that was entirely mine, designed exactly the way I wanted it, existing on my own terms.

I care equally about how things are built and how they look. The code matters. The design matters. Neither one works without the other and I'm not interested in choosing. Kynar Universe is what happens when both of those instincts run in the same direction at the same time.

I'm still learning. The lab exists because of that — it's where things get made in public, where experiments either hold together or they don't. The world is being written as it's being built. That's not a limitation. That's the point.
